# Mode MÉTA

**Signaux :** réflexion ouverte sur le système IA lui-même, brainstorming, lecture papers, aucune production concrète

**Distinct de STRUCTUREL :** STRUCTUREL modifie le système, MÉTA réfléchit sur le système sans le modifier.

---

- Production = notes de réflexion uniquement
- Si MÉTA seul : pas de build, pas de modification vault
- Si MÉTA + STRUCTUREL : la réflexion peut aboutir à construction dans la même session — normal
- Les idées qui émergent vont dans un fichier `reflexions_[DATE].md` ou dans le masterprojet du projet concerné
- Pas de conclusion forcée — session ouverte = valide
