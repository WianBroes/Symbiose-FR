# Agent-writable memory — consolidation automatique

**Palier :** 💡 IDÉE
**Créé :** 2026-06-04

## Principe
Un sous-agent qui consolide `observations.md` entre les sessions — comme le
sommeil REM (AutoDream system, fév. 2026).

## Application à AiBrain
Entre les sessions, un subagent (worker) pourrait :
1. Lire `observations.md`
2. Vérifier quelles observations ont atteint 3 confirmations
3. Proposer la graduation dans `CORE.md` section 2 ou la Cartographie
4. Marquer les observations graduées

## Pourquoi
- Automatise le pipeline épistémique sans intervention de Wian
- Utilise les subagents maintenant opérationnels
- Wian valide, l'IA prépare
